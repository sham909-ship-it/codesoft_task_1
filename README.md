# Task Title:Number-guessing game
# Language used:Java

## Description
-This is a simple console-based number game written in Java.
-This program generates a random number -It offers us to guess numbers between 1 to 100.
- if the guess and generated random number matches by which the game ends. 

## what the user does
-The user gives their guesses on which number the program could have generated 
-The game continues until the user enters or guesses the correct generated number by the program

## Features of the program
-Random Number generation
-User input using Scanner
-Too high/too low hints 
-Number of Attempts is counted
-Performance feedback based on attempts

## How to Run the Program
-Make sure Java is installed
-compile the java file with: javac filename.java
-Run the program using terminal
java filename.java
